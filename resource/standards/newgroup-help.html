Create New Group or Sub-Group

In JPasswords new groups are created by editing field GROUP for a database entry. Sub-groups are separated from their ancestors by a dot, so for example "Internet.Provider.Abroad" for a nested sub-group "Abroad" under "Provider" under "Internet". Groups have no independent nature and exist only as long as they are noted in at least one of the entries.

You can edit a group value directly in the entry editor or for a set of selected entries over function "Move to Group", available through a right-click. - Try it out! The "Undo"-function lets easily correct any mistakes.
