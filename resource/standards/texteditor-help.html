Shortcut Keys in Text Editor

Content Assignments

CTRL-D Insert current date (e.g. "2011-12-18 ")
CTRL-T Insert local time (e.g. "2011-12-18 22:12:36 ")
CTRL-U Insert time UTC (e.g. "17:12:36 UT ")

Selection Keys

CTRL-W Select next word
CTRL-L Select current line
CTRL-P Select paragraph
CTRL-A Select all

Editing Keys

CTRL-Z Undo last action
CTRL-Y Redo last action
CTRL-X,
SHIFT-del 		Cut (Clipboard)
CTRL-C,
CTRL-ins 		Copy (Clipboard)
CTRL-V,
SHIFT-ins 		Paste (Clipboard)
