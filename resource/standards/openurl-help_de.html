URL-Datei-Verbindung

Allgemein

Mit der URL-Datei-Verbindung sind Sie in der Lage, Passwort-Datenbanken zu öffnen, die sich auf einer entfernten Ressource befinden und über eines der Internetprotokolle HTTP oder FTP erreichbar sind. Über HTTP geladene Dateien können nur gelesen, aber nicht gespeichert werden. Über FTP geladene Dateien können auch gespeichert werden; die Anlage neuer Dateien is möglich, wenn die Programmoption im Tab "Sicherheit" freigeschaltet ist. FTP mit SSL ist derzeit nicht möglich.

Sicherheitsbedenken

Generell ist es sehr gefährlich, sicherheitssensible Daten auf einer öffentlich zugänglichen Ressource zu hinterlegen, wie gut die verwendete Verschlüsselungstechnik auch immer sein mag. Diese Vorrichtung wurde für Leute geschaffen, die sie unbedingt brauchen oder die möglicherweise ein privates Netz benutzen, das sie für sicher genug halten.

Falls Sie eine Passwort-Datenbank auf eine Internet-Ressource legen wollen, sollten Sie unbedingt darauf achten, die bestmöglichen Sicherheitseinstellungen für die Datei zu verwenden. Das Zugangspasswort (geheimer Schlüssel) sollte nach meiner Überzeugung ein zufallsgenerierter Schlüssel von mindestens 12 Zeichen Länge sein. Worte aus dem Wörterbuch müssen unbedingt vermieden werden!

Kontrollelemente

Das editierbare Auswahlfeld URL akzeptiert gültige URL-Ausdrücke. Das Feld erinnert darüber hinaus bis zu 8 erfolgreich durchgeführte Aufträge, wenn die Option "Liste der zuletzt geöffneten Dateien verwenden" in den Programm-Optionen aktiviert ist.

Die Löschen-Schaltfläche entfernt alle Einträge aus der URL-Auswahlliste.

Durchführung

Die OK-Schaltfläche startet die Durchführung und schließt diesen Dialog. Abhängig von Ihrer Internet-Verbindung kann es einige Sekunden dauern, bis eine Antwort angezeigt wird. (Sie können den Arbeitsstatus an dem kleinen Radzeichen in der rechten unteren Ecke des Programmfensters erkennen.)

Wenn Sie eine FTP-Adresse eingegeben haben, wird die folgende Antwort wahrscheinlich eine Login-Anfrage der FTP-Liegenschaft sein. Geben Sie Benutzername und das Passwort in dem Dialogfeld ein. Zu Ihrem Komfort erinnert JPasswords die FTP-Logins für die Dauer einer einzelnen Programmsitzung.

Nachdem eine Verbindung zu der Datei aufgebaut wurde, erfolgt die Passwort-Abfrage und die Datenbank kann in den Arbeitsbereich geladen werden.
